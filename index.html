How the web works

The web is a bunch of computers that communicate with each other. When person goes to a web page like www.yahoo.com, their computer sends an HTTP request to a server.The server finds the appropriate HTML document and sends it back to the user's computer where a web browser interprets the page and displays it on the user's screen.Click here

HTML Hypertext Markup Language

HTML is made up of the following
-Text Content
-Mark-up (what it looks like)
-References to other Documents (e.g. images, and video)
-Links to Other Pages

HTML Mark-up


The first tag used is the opening tag followed by content, ending in a closing tag.

The entire HTML line (Opening tag-Content-/Closing Tag) is referred to as an Element. Failure to us the closing tag will fail to end the mark-up effect. The remaining part text will be bold or Italics until the closing tag is used.


Mark-up Tags

< b > Bold < /b > BOLD
< em > Emphasize ( Italics)

The < img > tag is used to link to images

To link to an image your command format will look like this. You will use two additional fields the first is src or ( source), the second is alt ( alternate). The source refers to the URL of the image, while the ALT field refers to the displayed text if the image fails to load,- it is required.There is no closing tag for an image, the command ends in a "void"tag that has no content- therefor void.

INLINE / Block

The < br > tag is inline, so br simply ends a line and wraps to the next line down. The < p > tag actually makes an invisible box around the text. The dimensions of the box can be adjusted height, and width. Inline Elements are < b > , < em > , < img >. and the < p > is the only block element learned so far.
